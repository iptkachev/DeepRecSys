# Week 7
## Sequential Recommenders 

Lecture by [Kirill Khrylchenko](https://github.com/KhrylchenkoKirill)

Seminar by [Vladimir Baikalov](https://github.com/NonameUntitled)

Youtube:
* [Lecture](https://youtu.be/LATI_HsAnkk)
* [Seminar](https://youtu.be/7hYJqQkWfMs)

In week 7 of the course, we move on to sequential recommenders — models that directly model user interaction history.

The lecture includes:
1. Transformers for modeling user history in ranking
2. Transformers for retrieval and representation learning

During the seminar, we discuss two classical papers on user modeling at Pinterest - [PinnerFormer](https://arxiv.org/abs/2205.04507) and [TransAct](https://arxiv.org/abs/2306.00248).
